# ProjectZS
## 프로젝트 개요
제목: Zombie Invasion<br>
장르: 3D FPS<br>
목표: 쏟아져 나오는 좀비들을 물리치고 최대한 오래 살아남기

## 유니티 버전
2021.3.8f1

## 실행 방법
bin 폴더에 있는 projectZS.exe 실행

## 조작 방법
이동: W, A, S, D<br>
사격: 마우스 좌클릭, Ctrl<br>
조준: 마우스 우클릭, Alt<br>
재장전: R<br>
점프: 스페이스 바<br>
엎드리기: C<br>
무기 교체: 1, 2<br>
